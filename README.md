## App Overview & Architecture
This application is an elite **Artisanal Coffee Shop Management Catalog** built utilizing a structural **Model-View-Controller (MVC)** framework architecture.

* **Model Layer:** `CategoryModel` and `CoffeeModel` encapsulate structured data fields ensuring safe property parsing.
* **View Layer:** Implements responsive material design rendering clean lists, a side Navigation drawer menu interface, deep product details inspection views, and developer metadata panels.
* **Controller Layer:** `ShopController` acts as state processor managing lookups, search text filtering criteria processing, and dynamic runtime system settings.

---

## Mandatory Complex Features Built
1. **4 Structured Screens:** Dashboard View, Item Grid Category Filter View, Detailed Variant Profiler, and Developer Bio layout view.
2. **Dynamic Drawer Routing:** Sidebar Navigation Drawer built with custom user header elements.
3. **Lazy-List Generation:** Utilizes `ListView.builder` linking context dynamically against controller instances.
4. **Data Injection Parsing:** Clean object-based data routing between primary listings and detail view layouts.
5. **Unique Feature Applied:** Integrated dynamic system interface Dark/Light display scheme toggle running inside global entry parameters via the Controller.
6. **External Packages & Local Assets:** Utilizes package `google_fonts` rendering layout topography together with image asset mapping assets.
